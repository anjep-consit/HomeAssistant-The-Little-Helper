# Maskotbilleder til Den Lille Hjælper

Læg disse PNG-filer i denne mappe i din Home Assistant config:

- `mascot_idle.png` - normal og glad
- `mascot_thinking.png` - når en reminder venter
- `mascot_celebrate.png` - når en milepæl er nået
- `mascot_comfort.png` - når dagens humør er lavt
- `mascot_sleeping.png` - før kl. 06:00

Anbefalet størrelse: 256 x 256 px, transparent baggrund. En venlig papirclips eller minimalistisk hjælper-figur passer godt til temaet.
